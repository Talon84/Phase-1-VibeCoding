# Fase 1 do Projeto Integrador


### 1. Definição do Problema

Muitos estudantes têm dificuldade em organizar tarefas da escola ou faculdade, como trabalhos, provas e atividades. Muitas vezes os prazos acabam sendo esquecidos, causando atrasos ou acúmulo de tarefas.

Para ajudar nesse problema, o projeto propõe a criação de um **sistema simples chamado _EZTask_**, que ajudará estudantes a registrar tarefas, acompanhar prazos e organizar melhor seus estudos.

### 2. Criação de Personas com IA

As personas representam os possíveis usuários do sistema.

**Persona 1 – João**

21 anos
* Estudante universitário
* Dificuldade em lembrar prazos de trabalhos.

**Persona 2 – Ana**

27 anos
* Trabalha e estuda
* Precisa organizar melhor o tempo para estudar.

**Persona 3 – Lucas**

* 17 anos
* Estudante do ensino médio
* Tem muitas atividades e precisa de ajuda para organizar tudo.

### 3. Proposta de Valor

O EZTask pretende ajudar estudantes a organizar suas tarefas de forma simples.

Com o sistema será possível:

* Registrar tarefas
* Definir prazos
* Marcar atividades como concluídas
* Visualizar todas as tarefas em um só lugar
* Isso ajudará os estudantes a se organizarem melhor e evitar esquecer atividades importantes.

### 4. Lista de Funcionalidades

**Funcionalidades principais do sistema:**

* Cadastro de usuário
* Login no sistema
* Adicionar tarefas
* Definir prazo de entrega
* Marcar tarefas como concluídas
* Visualizar lista de tarefas

**Funcionalidades futuras**:

* Notificações de prazos
* Organização por disciplina
* Calendário de atividades
